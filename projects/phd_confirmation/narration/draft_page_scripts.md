## Page 0 - Cover

In this presentation, I will introduce the variable-wavenumber approximation, or VWA. The aim is to model nonlinear wave groups efficiently, while keeping the bound-wave physics visible and making higher-order harmonic estimation computationally practical.

## S0WhyNonlinearWavesSlides

### Page 1 - S0 / 1

I will start with the basic motivation: nonlinear waves matter because their shape changes first, and the timing errors accumulate later. This section uses one focused wave group to show both effects.

### Page 2 - S0 / 2

Here we first set up the linear reference. In physical space we have a localized wave group, and in spectral space it is represented by a compact peak around the carrier wavenumber.

### Page 3 - S0 / 3

Now we add the nonlinear correction to the same group. The main visible change is the shape: the crest lifts by about ten percent and the troughs become flatter.

### Page 4 - S0 / 4

In the spectral domain, we can see where that shape change comes from. The nonlinear field adds extra spectral energy near set-down, the second harmonic, and the third harmonic.

### Page 5 - S0 / 5

Next we place a fixed gauge in the wave field. As the group passes the gauge, the spatial difference becomes a time record, so we can compare what the instrument would actually measure.

### Page 6 - S0 / 6

The nonlinear group also has a small speed change, and that becomes a visible arrival drift at the gauge. So the correction is not only about crest height; it also affects local timing.

### Page 7 - S0 / 7

The takeaway is that nonlinear corrections affect both shape and timing. They sharpen and flatten the wave profile, and they shift what a fixed gauge records.

### Page 8 - S0 / 8

So the natural next question is: what are the extra components that created this change? I will now describe them as bound harmonics that travel with the group.

## S1BoundHarmonicsSlides

### Page 9 - S1 / 1

Now I will ask what nonlinearity added to the free wave group. The answer is a set of extra components that are generated by interactions and remain tied to the group.

### Page 10 - S1 / 2

We start again from the linear theory. At this stage the wave group is built from free-wave components, and there are no bound harmonics in the spectrum yet.

### Page 11 - S1 / 3

When nonlinear interactions are included, the bound part is built from several components. The first one appears near zero wavenumber: this is the set-down contribution, generated by the group rather than travelling as an independent free wave.

### Page 12 - S1 / 4

Next, the second harmonic appears near twice the carrier wavenumber. This is the first superharmonic correction, and it is one of the main terms responsible for sharpening the crest and changing the surface shape.

### Page 13 - S1 / 5

Then higher harmonics continue the same pattern. They are usually smaller, but they become important when the group is strongly focused or when we want high-order accuracy.

### Page 14 - S1 / 6

Putting these together, the nonlinear surface is the free wave plus the bound components. This is the decomposition I will use throughout the rest of the presentation.

### Page 15 - S1 / 7

In physical space, those bound terms reshape the group. They lift the crest, flatten parts of the profile, and change the local wave shape without becoming separate free waves.

### Page 16 - S1 / 8

As the group moves, the bound components move with it. They stay inside the envelope because their phases are slaved to the free-wave components that generated them.

### Page 17 - S1 / 9

This is the key point: bound harmonics are not independent free waves. They remain inside the wave group that generated them, so the modelling task is to calculate them accurately and efficiently.

## S2ExactInteractionsSlides

### Page 18 - S2 / 1

Now let us look at why exact interaction theory becomes expensive. Bound harmonics require interactions between spectral components, so the cost depends on how many components we retain.

### Page 19 - S2 / 2

At first order, the calculation is still simple. Each retained component contributes independently to the linear spectrum, so there is no interaction table to build.

### Page 20 - S2 / 3

At second order, the output spectrum is built from pairs. Each pair contributes to a sum wavenumber, so the calculation changes from independent components to pairwise interactions.

### Page 21 - S2 / 4

In the exact second-order formula, every pair carries an amplitude product, a kernel value, and the combined phase. This keeps the component-resolved physics, but it requires a double sum.

### Page 22 - S2 / 5

As those pair contributions accumulate, the second-order spectrum fills in. The exact summation is accurate, but it becomes expensive because every retained pair must be evaluated.

### Page 23 - S2 / 6

The timing data shows that this exact MF12 summation is already the bottleneck. The compact evaluation is several orders faster, even before we move to higher order.

### Page 24 - S2 / 7

Directional groups make the exact cost explode further. Once spreading and retained-energy thresholds are included, the number of retained directional components can become very large.

### Page 25 - S2 / 8

Here the practical gap is visible. MF12 grows from seconds to hours as more directional components are retained, while the VWA calculation stays near a small fixed cost on this scale.

### Page 26 - S2 / 9

This motivates the central question. Can we keep the physics without evaluating every pair explicitly? The answer I introduce next is to reduce the interaction kernel using VWA.

## S3VWAStructureSlides

### Page 27 - S3 / 1

In this section, I will introduce the VWA idea: replacing explicit pairwise kernel evaluation with a compact spectral-product form. The goal is to keep the interaction phase structure, while avoiding the full component-pair lookup.

### Page 28 - S3 / 2

Let us start from the exact second-order sum. Every retained pair contributes an amplitude product, an exact interaction kernel, and the combined phase of the two linear components.

### Page 29 - S3 / 3

Each term in the sum has two jobs. The phase tells us where the bound wave appears, while the amplitude factor tells us how strongly that pair contributes.

### Page 30 - S3 / 4

Exact theory repeats this same operation for every component pair. That is physically precise, but it is also the computational pattern we want VWA to avoid.

### Page 31 - S3 / 5

Here we can view the exact kernel as a two-index lookup table over the two interacting wavenumbers. One off-diagonal pair needs one entry of this table, and the number of entries grows quadratically with the retained spectrum.

### Page 32 - S3 / 6

The first thing we want to preserve is the phase. Each pair fixes a summed phase, and therefore a summed bound wavenumber. Different pairs still generate different bound-wave phases.

### Page 33 - S3 / 7

VWA keeps that variable phase, so the generated bound wave still appears at the pair-sum wavenumber. The approximation enters through the amplitude kernel, which we reduce to a one-index function.

### Page 34 - S3 / 8

We replace the two-index exact kernel using self-interaction samples on the diagonal. In other words, VWA estimates an off-diagonal pair by reusing one-index kernel information at the two participating wavenumbers.

### Page 35 - S3 / 9

This is the key VWA step. Because the two pair indices are interchangeable in the second-order sum, we can use a symmetric approximation for the off-diagonal kernel.

### Page 36 - S3 / 10

Instead of using the full two-variable kernel, VWA replaces it by the average of two self-interaction values: one taken from each participating wavenumber.

### Page 37 - S3 / 11

So the off-diagonal interaction is represented by one half times the sum of the two diagonal self-kernel samples. The amplitude is approximated, but the phase is still the exact pair phase.

### Page 38 - S3 / 12

After substituting this symmetric average back into the sum, the expression separates into two matched products. That is why the final VWA formula no longer needs a full two-dimensional kernel table.

### Page 39 - S3 / 13

Computationally, this means we only need a one-index kernel. We weight the input spectrum once, take inverse FFTs, and multiply the resulting physical-space signals.

### Page 40 - S3 / 14

This is why the VWA form is fast. The cost is dominated by a small number of FFTs and pointwise products, rather than by explicitly looping over all retained spectral pairs.

### Page 41 - S3 / 15

The contrast becomes clearer as the number of spectral components grows. The exact second-order expression accumulates many unique interactions, while the VWA expression keeps the same compact product form.

### Page 42 - S3 / 16

So after the phase and cost arguments, the remaining question is amplitude accuracy. VWA has avoided the exact summation, but we still need to check how good the kernel approximation is.

### Page 43 - S3 / 17

We test that directly with the split kernel comparison. The upper triangle shows the exact MF12 reference, while the lower triangle shows the VWA kernel on the same two-wavenumber plane.

### Page 44 - S3 / 18

Kernel error only matters if it changes the reconstructed wave field. So here we check the surface elevation in a deep-water, narrow-band case where the approximation should be strongest.

### Page 45 - S3 / 19

The second waveform comparison stresses the method more, using finite depth and broader bandwidth. This is where we expect amplitude differences to become easier to see, especially near the crest and side lobes.

### Page 46 - S3 / 20

To avoid judging only by eye, I use a spectrum-level error metric called Q. A value near zero means the model spectrum is close to the exact reference, while larger values indicate more mismatch.

### Page 47 - S3 / 21

Finally, the Q sweep shows how the approximation changes with bandwidth and depth. This gives a more systematic view than a single waveform: VWA performs best where the kernel varies gently, and the error grows when the interaction kernel changes more strongly.

## S4HigherOrderVWASlides

### Page 48 - S4 / 1

Now I will extend the same VWA idea to higher order. The question is whether the phase structure and the compact FFT-ready product form still make sense beyond second order.

### Page 49 - S4 / 2

Second order showed the key mechanism. But bound waves do not stop at second order, so the next step is to ask what changes when we go to third, fourth, and fifth order.

### Page 50 - S4 / 3

The exact theory adds one more wavenumber at each higher order. So the second-order pair becomes a third-order triplet, then a fourth-order tuple, and so on.

### Page 51 - S4 / 4

This is where the cost grows quickly. Order two scales like \(N_c^2\), order three like \(N_c^3\), and by order five the exact component-resolved interaction count is already \(N_c^5\).

### Page 52 - S4 / 5

The important phase structure, however, is still exact. At order \(n\), the bound harmonic carries the sum phase and sum wavenumber of the participating components, not a new independent free-wave phase.

### Page 53 - S4 / 6

So VWA keeps that same summed phase structure, but replaces the \(n\)-wavenumber kernel by one weighted signal. This gives a compact expression for the order-\(n\) bound component.

### Page 54 - S4 / 7

Here we can see the same formula written for different orders. The second-, third-, fourth-, and fifth-order terms all follow the same pattern: one kernel-weighted field times powers of the linear field.

### Page 55 - S4 / 8

This is the practical computational comparison. Exact order \(n\) needs all wavenumber tuples and scales like \(N_c^n\), while the VWA order-\(n\) form remains FFT dominated.

### Page 56 - S4 / 9

Once the formula is compact, we still need to check the result. The key question is whether the ordered bound harmonics remain coherent when we move from second to fifth order.

### Page 57 - S4 / 10

This figure shows the unidirectional surface-elevation result from second to fifth order. The comparison is made against fully nonlinear simulation, so we can check whether the ordered harmonics have the right structure.

### Page 58 - S4 / 11

Across the panels, the VWA harmonics remain organized with the focused group rather than becoming arbitrary corrections. This is the evidence that the compact higher-order structure is behaving sensibly.

### Page 59 - S4 / 12

So the message of this section is that the same VWA structure extends from second to fifth order. We keep the summed phase structure, use a compact weighted field, and avoid the exact high-dimensional tuple sums.

### Page 60 - S4 / 13

With this higher-order route in place, we can now move from the surface elevation itself to related applications. In the final section, I will use the same compact structure for directional extension, time series, inverse recovery, and surface kinematics.
